# counterdev
counterdev - dev версия шахматного движка [Counter](https://github.com/ChizhovVadim/CounterGo). Текущая версия работает на arm64 neon (Apple silicon) с использованием simd инструкций для нейросетевой оценочной функции.

## Утилиты
Проект содержит утилиты, которые используются при разработке движка, но не нужны для запуска самого движка.
- решение тактических тестов
- проведение матча между двумя движками
- вычисление ошибки оценочной функции на валидационном датасете
- генератор дебютов
- создание датасета для обучения нейронной сети
- обучение нейронной сети
